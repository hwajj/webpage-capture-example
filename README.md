### 웹서버 관련

- 웹서버 실행

```
npm run start
```

- sass -> css 자동 변환

```
sass --watch src/css/style.scss:src/css/style.css
```

### 웹팩관련

- 웹팩 css 설정
  - scss-loader : scss를 css로 컴파일
  - css-loader : css를 js로 변환
  - style-loader : style태그에 js넣어주고 head 태그안에 style태그 넣어줌
